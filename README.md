# Abby Career OS

A mobile-first personal career operating system for Precision Health, Healthcare AI, Clinical Data Analytics, and Precision Medicine careers.

## Run locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## GitHub Pages

This repo includes `.github/workflows/pages.yml`. Push to `main`, then enable GitHub Pages with **Source: GitHub Actions** in the repository settings.

## PWA

The app includes a web app manifest, install icons, and a service worker. After deployment, open the GitHub Pages URL on iPhone Safari and use **Share -> Add to Home Screen**.

## Learning Plan

The Learning Center ships with clean weekly course templates for SQL, Python, Pandas, Statistics, and Machine Learning. Each weekly task can be checked off manually, or the **Next** button completes the next open task.

Use the reset button in the app header to clear local data and return to a clean template.
